# Avalanche Quest 한국어 상세 명세서

이 디렉터리는 게임, 프론트엔드, 백엔드, Web3, 컨트랙트, 데이터, 보안,
테스트와 운영의 구현 계약을 담는다. 루트 `AGENTS.md`와 `docs/agents`의
규칙을 함께 적용하며 보안 규칙을 약화할 수 없다.

## 사용법

변경 전 해당 도메인의 README와 영향받는 모든 명세를 읽는다. 코드와 문서가
다르면 임의로 하나를 선택하지 말고 차이를 해결한다. 상태가 `기준선`인 문서는
즉시 적용되는 규칙이며, 실제 파일·테스트 링크가 채워져야 `구현됨`이 된다.

## 확장 설계

아직 구현되지 않은 교차 도메인 확장안은 기존 120개 기준선과 구분해 관리한다.

- [NFT 진행 시스템](../nft-system/README.md): 장비 loadout, 클래스 NFT, 전직,
  ERC-1155 재료, V2 컨트랙트와 마이그레이션
- [게임 콘텐츠 제작·레이드 시스템](../game-content/README.md): 캐릭터 frame,
  anchor/socket, 스킬 icon/VFX, 일반 몬스터, stage encounter와 raid boss pattern

확장 설계를 구현할 때 영향받는 기존 도메인 명세를 함께 갱신하며, 구현 전 문서를
현재 동작으로 표현하지 않는다.
