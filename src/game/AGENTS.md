# 게임 코드 에이전트 규칙

이 규칙은 `src/game` 전체에 적용한다. 루트 규칙과 `docs/specs/02-게임-코어`,
`03-캐릭터`, `04-스킬`, `05-적과-보스`, `06-에셋과-애니메이션`을 먼저 읽는다.

- Phaser 코드는 지갑·viem·wagmi·ABI·DB·서버 모듈을 import하지 않는다.
- 게임 세션 입력은 직렬화 가능하고 버전이 있으며 실행 중 불변이어야 한다.
- 보상액, NFT 종류와 mint 권한은 게임 코드가 결정하지 않는다.
- 새 스킬과 보스 패턴을 `QuestScene`의 거대한 조건 분기로 추가하지 않는다.
- 시간은 Phaser clock과 `delta`를 사용하고 프레임 수에 게임 결과를 의존하지 않는다.
- 투사체와 VFX는 풀링하고 생성 상한과 수명, shutdown 정리를 정의한다.
- 상태 전이, 피해, 쿨다운, 다단 히트와 사망은 결정적인 테스트를 작성한다.
- 변경 후 lint, typecheck와 관련 게임 테스트를 실행한다.
